this is a stupid language to construct a 4d transformation matrix
you specify a plane of rotation and it's angle. You can use variables to
animate the expression.

example: zw(1/4) yw(1/4) xz(1/3 t) yz(1/4)

syntax: plane(expr var?) ..
planes: zw yw yz xw xz xy
vars: t (= time) x y (= mouse)
